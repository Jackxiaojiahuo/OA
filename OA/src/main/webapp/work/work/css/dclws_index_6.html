【项目背景-南通工商局简介】

南通市工商行政管理局为市政府工作部门之一，主管全市市场监督管理和行政执法。本机关共设13个内设机构，分别是：办公室、人事教育处、财务处、政策法规处、行政服务处、行政执法协调指导处、反垄断与反不正当竞争处、个体私营经济和企业监督管理处、市场规范管理处、合同监督管理处、商标监督管理处、广告监督管理处、消费者权益保护处，另设机关党委。
 
问题也由此而来。于内，复杂的组织机构及人员管理让工商局领导头痛；于外，庞大的市场监管和行政执法没有一个统一的内部平台来调配。不仅如此，近期为响应国家政策，工商局决定全面加强基层建设，落实 “四化”建设要求, 推进CIS科学管理, 提高工作运转效率, 实现科学绩效考核，因此迫切需要一套完全对路的软件系统，将监督管理、规范工作和绩效考核落到实处，以便充分调动基层工作的积极性，激发干部队伍的工作潜力。

【实施过程】

南通工商局经过长时间比较和验证，最终选择10oa协同办公系统，以绩效考核为主线，重点打造集工作目标管理、工作过程控制和工作结果监督于一体的高效管理平台。该平台不仅包含了传统OA的公文起草、核准和下发环节，更包含了执行、分解、反馈、监督和考核环节，实现了工作任务的全过程自动化处理。
 
实施具体内容如下： • 制定政务流程：利用 10oa 核心工作流引擎、图形化流程定制能力，快速创建和优化党政机关政务流程（包括收发文流程）。自动分发数以万计的电子化任务（领导指示、往来公文、请示汇报等）到相关人员的桌面，并记录全部的执行过程。辅以 10oa 电子邮件和即时通信，各级领导和办公人员流畅协作，时空无阻，大幅提升全员工作效率。
• 效能监察：专门的效能监察模块，可以从多个角度查看任务的状态，监控任务的执行情况，并能够给不正常的任务标注上蓝色预警、黄色预警甚至红色预警。专门的绩效考核模块，能够量化考核指标并层层分解，与工作任务完美结合，将考核落实到每一项工作中，从而积累真实、准确的基础考核数据，并自动计算出一段期间累计的考核结果。
• 政务资源精细管理：利用 10oa 出色的信息组织、编辑、存储和分享能力，对档案、台帐等政务资源进行电子化、精细化管理，大大减少手工操作的繁琐和人为失误的可能。
• 分层管控和协同：在具有多个下属机构，或者需要进行跨机关的公文和任务流转的情况下，可以部署多套分支系统。每个系统支持独立运行，自主管理，支持内部的无缝协作，满足自身的政务需求。通过 10oa Multi-branches 强劲的政务网关引擎，定义跨系统的政务运作规则，自动地跨系统传递领导指示、往来公文、请示汇报等多种任务，并全面记录政务运转过程。
•安全管理：集中存储、身份识别、功能授权、数据授权等多种安全机制，保障信息的可靠性和安全性。整个系统可以在政务专网上运行，与公众外网相隔离，完全避免病毒袭击和非法访问。



【实施效果】

10oa协同办公系统的成功实施为南通工商局的管理工作带来了显著变化：
 • 政务流程：从单向流转向一体化流转转变，适应实际工作中任务各种复杂的流转情况。
 • 监督检查：从侧重事后检查转向实时检查转变，避免无记录、缺监督的问题，加快政务处理进度。
 • 绩效考核：从总体评价向定量分析转变，对单位和个人进行全面分析和客观、公正、量化的考核。
 • 资源管理：从粗放管理向精细管理转变，解决档案台帐种类多、无纸化办公没有真正实现的问题。
 
总体上，此次合作为南通工商局实现了信息资源全面共享、建立了相互沟通协同的工作平台、加快了工作任务运转效率、实现了远程全过程的监督管理、提高了绩效考核的科学性和全面性，加快了工作任务运转效率，真正达到“无纸化”办公的效果，多次获得江苏省局和国家总局的支持和表扬。
